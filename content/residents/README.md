# 居民討論紀錄 (Residents' Discussion)

本部分收集整理宏福苑居民在火災期間及之後的討論、互助信息及社區聲音。這些記錄對於還原事件經過、了解居民需求以及社區互助情況具有重要參考價值。

## 內容概覽

目前正在整理各個渠道的居民討論，包括但不限於：

- **即時通訊群組**：WhatsApp, Telegram 等業主/居民群組的互助信息
- **社交媒體討論**：Facebook 群組、討論區的相關帖文
- **線下討論**：居民大會、關注組會議的相關記錄
